# 코디세이 프로그래밍 과제 저장소

각 과제는 과정 번호와 문제 번호를 기준으로 폴더를 구분하여 관리합니다.

## 문제 01  
미션 컴퓨터 로그를 분석하여 사고 원인을 파악하는 과제

**구현 내용**

- Python을 이용한 로그 파일 분석
- 로그 역순 정렬 출력
- 문제 로그 별도 파일 저장
- 로그 분석 보고서 작성

---

## 문제 03
화성 기지 적재 화물 목록을 분석하여 인화성이 높은 물질을 분류하는 과제

**구현 내용**

- `Mars_Base_Inventory_List.csv` 파일 읽기 및 출력
- CSV 데이터를 Python 리스트 객체로 변환
- 인화성 지수(`Flammability`) 기준 내림차순 정렬
- 인화성 지수 0.7 이상 항목 별도 추출 및 출력
- 위험 물질 목록을 `Mars_Base_Inventory_danger.csv` 파일로 저장
- 정렬된 목록을 `Mars_Base_Inventory_List.bin` 파일로 이진 저장
- 저장된 이진 파일을 다시 읽어 출력
- 텍스트 파일과 이진 파일의 차이점 및 장단점 보고서 작성

---

## 문제 06
화성 기지 환경 정보를 확인하기 위한 더미 센서를 구현하고,  
환경 값을 생성·출력·기록하는 과제

**구현 내용**

- `DummySensor` 클래스 생성
- `env_values` 사전 객체를 멤버로 추가
- `set_env()` 메서드를 통해 환경 값을 랜덤으로 생성
- `get_env()` 메서드를 통해 환경 값 반환
- `DummySensor` 클래스를 `ds`라는 이름으로 인스턴스화
- `set_env()`와 `get_env()` 호출 결과 확인
- 전체 코드를 `mars_mission_computer.py` 파일로 저장
- 환경 정보를 날짜와 시간과 함께 로그 파일로 저장하는 기능 추가
- 과제 수행 내용을 보고서로 작성
